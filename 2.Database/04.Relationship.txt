- suppose we have users courses and notes  so how do we connect a user with their notes and course .. so there are two ways.

- in mongodb we mainly use:
1. Embedded Documents
2. Referencing (ObjectId)

1. Embedded Documents: store related data inside the same documents
- suppose an order contains product
- instead of storing products seperately we store them inside the same document
- ex: 
{
    "id":"ud",
    "name":"nova",
    "notes":[
      {
         "title": "JWT",
      "content": "Authentication"
      },{
      "title": "MongoDB",
      "content": "NoSQL"
      }
    ]
}

//models/course.js

const mongoose=require("mongoose");
const lessionSchema=new mongoose.Schema({
    title:String,
    duration:Number,
});

const courseSchema=new mongoose.Schema({
    title:String,
    lessions:[lessionSchema] //embedded array
});

module.exports=mongoose.medel("Course",courseSchema);

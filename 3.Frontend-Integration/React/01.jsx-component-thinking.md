- jsx: html code written inside the js
- react convert jsx code into javascript behind the screen 

- component :js function that return jsx , reusable ui block

```
function Greet(){
    return <h1> hello</hi>
}
```
- component naming: Most start with capital letter
- bcoz react treat lowercase as html tags
- always return in parentheses if return have multiple lines of code

```
function App(){
    return( <div>
    <h1>Hii</h1>
    <p>welcome to user interface</p>
    </div>
    )
}

```


//using a component:
```
function Greet(){
    return(
    <div>
    <App>
    </div>
    )
} 
```
//this is nesting component: a component contain another component
